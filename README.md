# LogFood

Учебно-практическое React-приложение для учета питания по дням. LogFood помогает вести дневник еды, считать калории за выбранную дату и хранит данные локально в браузере.

Проект развивается как frontend MVP: сначала стабильный дневник калорий, затем расчет по граммам, цели, статистика и справочник продуктов.

## Возможности

- добавление записи еды с количеством калорий;
- валидация формы добавления;
- редактирование записи;
- подтверждение удаления записи;
- выбор даты дневника;
- просмотр записей за конкретный день;
- история дней с переходом на выбранную дату;
- подсчет общей суммы калорий за день;
- сохранение записей в `localStorage`;
- защита от ошибок чтения и записи `localStorage`;
- маршруты `/`, `/day/:date`, `/history`;
- fallback route для неизвестных адресов.

## Основные сценарии

- открыть дневник;
- выбрать дату;
- добавить запись еды;
- исправить запись;
- отменить редактирование;
- удалить запись через подтверждение;
- перейти в историю;
- открыть день из истории;
- обновить страницу и проверить, что записи сохранились.
