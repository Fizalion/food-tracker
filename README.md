# LogFood

Учебно-практическое React-приложение для учета питания по дням.

## Возможности

- добавление еды с количеством калорий;
- валидация пустых значений и калорий меньше 1;
- просмотр записей выбранного дня;
- подсчет общей суммы калорий за день;
- удаление записей;
- история дней с переходом на конкретную дату;
- сохранение записей в `localStorage`;
- маршруты для дневника и истории.

## Стек

- React;
- Redux Toolkit;
- React Redux;
- React Router;
- CSS Modules;
- TypeScript постепенно: тип модели `FoodEntry` и типизированные утилиты.

## Запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm run build
```

## Основные сценарии

- открыть дневник;
- выбрать дату;
- добавить запись еды;
- удалить запись;
- перейти в историю;
- открыть день из истории;
- обновить страницу и проверить, что записи сохранились.
