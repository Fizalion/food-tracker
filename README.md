# Food Tracker

Небольшое React-приложение для учета еды и калорий.

## Что умеет

- Добавлять название еды и количество калорий.
- Показывать общий итог калорий.
- Выводить список добавленных продуктов.

## Запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm run lint
npm run build
```
