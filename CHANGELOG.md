# Changelog

Все заметные изменения продукта будут описываться здесь.

Формат основан на Keep a Changelog, версии следуют Semantic Versioning.

## [0.6.0] - 2026-06-06

### Added

- Локальный справочник продуктов с калорийностью на 100 г.
- Подсказки продуктов при вводе названия еды.
- Выбор продукта из подсказок с автозаполнением названия и ккал на 100 г.
- Оформление выпадающего списка подсказок.

## [0.5.0] - 2026-06-03

### Added

- Отдельная страница недельной статистики.
- Недельные итоги: общая сумма калорий, среднее за день и дни с превышением цели.
- Список 7 дней с калориями, процентом от цели, статусом и progress bar.
- Пустое состояние для недели без записей.

## [0.4.0] - 2026-05-24

### Added

- Дневная цель калорий с хранением в `localStorage`.
- Donut-диаграмма прогресса к дневной цели.
- Отображение процента от цели, остатка калорий, выполненной цели и превышения.
- Redux slice для цели калорий и типизированная утилита расчета прогресса.

## [0.1.0] - 2026-05-02

### Added

- Первый рабочий прототип Food Tracker.
- Добавление еды с количеством калорий.
- Подсчет общего количества калорий.
- Список добавленных продуктов.
- Базовая валидация формы.
