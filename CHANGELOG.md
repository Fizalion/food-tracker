# Changelog

Все заметные изменения продукта будут описываться здесь.

Формат основан на Keep a Changelog, версии следуют Semantic Versioning.

## [0.4.0] - 2026-05-24

### Added

- Дневная цель калорий с хранением в `localStorage`.
- Donut-диаграмма прогресса к дневной цели.
- Отображение процента от цели, остатка калорий, выполненной цели и превышения.
- Redux slice для цели калорий и типизированная утилита расчета прогресса.

## [0.1.0] - 2026-05-02

### Added

- Первый рабочий прототип Food Tracker.
- Добавление еды с количеством калорий.
- Подсчет общего количества калорий.
- Список добавленных продуктов.
- Базовая валидация формы.
